ERROS DE ATIVAÇÃO — MAQUININHAS PAGBANK
========================================

USE ESTE DOCUMENTO quando o cliente relatar que está tentando ativar a maquininha
pela primeira vez ou reativar após troca, e recebe uma mensagem de erro.


ERRO A001 — ATIVAÇÃO CANCELADA
-------------------------------
Quando ocorre: a operação de ativação foi cancelada.
Na tela aparece: erro A001.

Passos para o cliente:
1. Oriente o cliente a realizar novamente o processo de ativação desde o início.
2. Se o erro persistir, pergunte se algum botão físico do teclado da máquina está
   com problema (preso, travado ou não responde ao toque).
3. Se o botão vermelho (cancelar) estiver com problema, o cliente deve verificar
   se é teclado touch ou físico e seguir o fluxo de teclado com defeito.
Escalação: se os passos acima não resolverem, abra ticket de suporte técnico.


ERRO M3021 — MÁQUINA JÁ ATIVADA EM OUTRA CONTA / PRECISA SER DESATIVADA
------------------------------------------------------------------------
Quando ocorre: a máquina já está vinculada a outro cadastro e precisa ser desativada
para poder ser ativada na conta atual.
Na tela aparece: erro M3021.

Passos para Moderninhas:
1. Peça ao cliente para acessar: Menu > Configurações Gerais > Configuração Remota.
2. Aguarde a máquina ser desativada automaticamente.
3. Após a desativação, realize a pré-ativação da máquina novamente na conta do cliente.
4. Solicite ao cliente que faça a ativação da máquina na mesma conta.

Passos para Leitores (Minizinha):
1. Desinstale o aplicativo PagBank no celular.
2. Reinstale o aplicativo.
3. Refaça o pareamento da máquina.

Escalação: se não conseguir desativar pelo menu, escale para o gerenciador interno.


ERRO M5000 — LEITOR OU CÓDIGO DE ATIVAÇÃO INVÁLIDO
---------------------------------------------------
Quando ocorre: o cliente comprou a máquina vinculada a um e-mail, mas está tentando
ativar em uma conta com outro e-mail. O sistema só permite a ativação se a máquina
estiver vinculada ao cadastro que está sendo usado.
Na tela aparece: erro M5000 — leitor ou código de ativação inválido.

Passos:
1. Pergunte ao cliente: "Você está usando o mesmo e-mail com o qual comprou a máquina?"
2. Se não: oriente o cliente a acessar a conta PagBank com o e-mail correto (aquele
   usado no momento da compra) e tentar ativar novamente.
3. Verifique se o código de ativação utilizado corresponde à conta correta.
4. O código de ativação pode ser obtido no aplicativo: Login > Vendas >
   Gerenciar Vendas > Maquininhas PagBank > Ativar Maquininha.

Escalação: se a conta não for identificada ou o cliente não tiver acesso ao e-mail
original, escale para a equipe de cadastro.


ERRO M5004 — LEITOR INATIVO
----------------------------
Quando ocorre: a Moderninha não está vinculada ao cadastro do cliente.
Na tela aparece: M5004 — leitor inativo.

Passos:
1. Verifique se a máquina está na lista de máquinas pendentes para ativação
   na conta do cliente: App > Login > Vendas > Gerenciar Vendas >
   Maquininhas PagBank > Ativar Maquininha.
2. Se a máquina estiver na lista: ajude o cliente no processo de ativação.
3. Se a máquina não estiver na lista: oriente o cliente a entrar em contato
   com o vendedor ou verificar com qual conta o dispositivo foi comprado.

Escalação: se a máquina não aparecer em nenhuma conta, abra ticket para
análise do cadastro do dispositivo.


ERRO M5005 — LEITOR INATIVO (SIMILAR AO M5004)
------------------------------------------------
Idêntico ao M5004. Siga o mesmo procedimento descrito acima para M5004.


ERRO M5007 — CÓDIGO DE ATIVAÇÃO DIGITADO ERRADO MAIS DE 3 VEZES
----------------------------------------------------------------
Quando ocorre: o cliente digitou o código de ativação incorretamente 3 ou mais vezes.
Na tela aparece: M5007 — código incorreto. Aguarde 30 minutos para tentar novamente.

Passos:
1. Informe ao cliente que o sistema bloqueou temporariamente por segurança
   e ele precisa aguardar 30 minutos antes de tentar novamente.
2. Após os 30 minutos, verifique se o cliente está usando o código correto:
   o código de ativação é encontrado no aplicativo PagBank, não no SMS.
3. Recomende fortemente que o cliente use a ativação por QR Code (mais fácil e
   sem risco de erro de digitação): abra o app > Ativar Maquininha > Escanear QR Code.

Escalação: se após 30 minutos e nova tentativa o erro persistir, abra ticket de suporte.


ORIENTAÇÃO GERAL PARA ERROS DE ATIVAÇÃO
-----------------------------------------
- Sempre confirme se o cliente está usando o aplicativo PagBank atualizado.
- Confirme se o e-mail e senha da conta PagBank estão corretos.
- Peça ao cliente para ter a máquina em mãos durante todo o processo.
- Se a máquina nunca foi ativada e chegou com erro, pode ser defeito de fábrica
  — neste caso, escale para análise de substituição.
